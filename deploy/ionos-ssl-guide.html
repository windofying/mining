🔒 IONOS SSL 配置指南

Mining Depot USA, Inc. - 完整SSL配置解决方案

⚠️ 重要提示

在开始配置之前，请确保您已经：

  • 拥有IONOS托管账户
  • 域名 mining-depot.com 已注册并指向IONOS
  • 获取了FTP/SFTP访问凭据
步骤 1: 激活SSL证书

1.1 登录IONOS控制面板

1. 访问 https://www.ionos.com

2. 使用您的账户凭据登录

3. 选择您的托管套餐

1.2 配置SSL证书

1. 进入"域名和SSL"部分

2. 选择域名 mining-depot.com

3. 找到"SSL证书"选项

4. 点击"激活SSL证书"

5. 选择"Let's Encrypt"免费证书

6. 确认激活

1.3 等待证书激活

• 通常需要5-30分钟

• 最多可能需要24小时

• 在等待期间不要修改域名设置

步骤 2: 检查域名配置

2.1 验证域名状态

1. 确认域名状态为"活跃"

2. 检查域名是否指向正确的服务器

3. 验证域名没有过期

2.2 检查DNS设置

使用在线工具验证DNS记录：

1. 访问 https://www.whatsmydns.net/

2. 输入 mining-depot.com 检查A记录

3. 输入 www.mining-depot.com 检查CNAME记录

4. 确保所有记录都指向IONOS服务器

步骤 3: 上传网站文件

3.1 准备文件

1. 运行部署脚本: deploy.bat

2. 确保所有文件都在 deploy\ 目录中

3. 检查 .htaccess 文件是否存在

3.2 上传到服务器

使用FTP客户端（如FileZilla）：

1. 连接到IONOS服务器

2. 导航到网站根目录（通常是 public_html 或 htdocs）

3. 上传 deploy\ 目录中的所有文件

4. 保持目录结构完整

3.3 设置文件权限

确保以下文件权限正确：

# 文件权限设置 .htaccess 文件: 644 (rw-r--r--) HTML 文件: 644 (rw-r--r--) CSS 文件: 644 (rw-r--r--) JS 文件: 644 (rw-r--r--) 目录: 755 (drwxr-xr-x)
步骤 4: 测试SSL配置

4.1 基本测试

1. 访问 https://www.mining-depot.com

2. 检查地址栏是否显示安全锁图标

3. 测试 http://mining-depot.com 是否重定向到HTTPS

4.2 使用诊断工具

1. 访问 https://www.mining-depot.com/ssl-test.html

2. 查看详细的SSL状态报告

3. 根据建议进行修复

4.3 在线SSL测试

1. 访问 SSL Labs测试

2. 输入 www.mining-depot.com

3. 查看SSL评级和问题详情

常见问题解决

问题1: SSL证书未激活

解决方案:

1. 在IONOS控制面板中重新激活SSL证书

2. 等待5-30分钟让证书生效

3. 检查域名是否正确解析

问题2: 重定向不工作

解决方案:

1. 检查.htaccess文件是否正确上传

2. 确认文件权限为644

3. 检查Apache mod_rewrite模块是否启用

问题3: 混合内容错误

解决方案:

1. 确保所有资源都使用HTTPS

2. 检查图片和CSS文件路径

3. 更新.htaccess文件

✅ 配置成功标志

当SSL配置成功后，您应该能够：

  • 访问 https://www.mining-depot.com 显示安全锁图标
  • 访问 http://mining-depot.com 自动重定向到HTTPS
  • 不再看到"不安全"或"ERR_SSL_PROTOCOL_ERROR"错误
  • SSL Labs测试显示A级评级
